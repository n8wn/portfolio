Ultimate Tic Tac Toe

A Java implementation of Ultimate Tic Tac Toe — a strategic variant where each move determines which board your opponent must play in next. Features a Monte Carlo Tree Search AI opponent that builds a statistical game tree through thousands of random simulations, balancing exploration and exploitation via the UCB1 formula. Supports Human vs Human, Human vs AI, and AI vs AI modes, with a built-in testing framework for benchmarking AI configurations across batches of games. Demonstrates MCTS, game tree search, object-oriented design, and statistical decision making.

Tech Stack: Java, OOP, Monte Carlo Tree Search, UCB1, Game Tree Simulation, Console I/O
